# Privacy Policy for GitHub Dev Shortcut

**Last updated:** December 28, 2025

## Overview

GitHub Dev Shortcut is a browser extension that toggles between github.com and github.dev. This policy explains how we handle your data.

## Data Collection

**We do not collect any data.**

This extension:
- Does NOT collect personal information
- Does NOT track browsing history
- Does NOT transmit any data to external servers
- Does NOT use analytics or tracking

## Data Storage

The extension stores only one piece of data locally in your browser:
- Your preferred keyboard shortcut (default: ".")

This is stored using Chrome's `storage.sync` API, which syncs across your Chrome browsers if you're signed in. This data never leaves your browser/Google account and is not accessible to us.

## Permissions

The extension requires:
- **storage**: To save your shortcut preference locally
- **Host permission (github.com, github.dev)**: To detect keypresses on GitHub pages only

## Third Parties

This extension does not share any data with third parties.

## Changes

Any changes to this policy will be reflected in the extension update notes.

## Contact

For questions about this privacy policy, please open an issue on the project's GitHub repository.
